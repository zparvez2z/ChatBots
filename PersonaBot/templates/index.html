Welcome!

You can use curl to test this API

To get this page

 curl http://chat-bot-url/

To Chat

 curl http://chat-bot-url/chat?query=hi

To Chat

 curl http://chat-bot-url/chat?query=hi

To get Chat History

 curl http://chat-bot-url/history

To get list of available persona

 curl http://chat-bot-url/persona

Set a Persona

To set a persona to chat with use PUT request like below

 curl http://chat-bot-url/set_persona/<int:persona_id> -X PUT -v

i.e : 

 curl http://chat-bot-url/set_persona/1 -X PUT -v

Create Persona

To create a persona first create a json file like below 

                  
                  {"persona":"my name is andrew .i like to code . i like python .i like data science . my favorite holiday is christmas.",
                  "back_story":" i started programming in 2015 . it is really fun and challenging. i started my programming using c.\
                                  now i code in python most of the time.",
                  "history":"i'm great today ! i'm reading a new book on deep learning"      
                  }

then send POST request like below 

 curl  -d @data.json -H "Content-Type: application/json" http://chat-bot-url/add_persona -X POST -v

Delete Persona

To delete a persona use DELETE request like below

 curl http://chat-bot-url/remove_persona/<int:persona_id> -X DELETE -v

i.e : 

 curl http://chat-bot-url/remove_persona/1 -X DELETE -v
